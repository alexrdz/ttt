# Terminal Tic Tac Toe

A tic tac toe game in the terminal using 

- [colors-cli](https://www.npmjs.com/package/colors-cli)
- [inquirer](https://www.npmjs.com/package/inquirer)

### Run the game

```
$ git clone git@github.com:alexrdz/ttt.git
$ cd ttt/
$ npm i -S
$ npm run start
```
